Комп'ютерні мережі, проміжний контроль   Нестор Олег, КНМС - 31

1) У сигналу 2.4Ghz більша дальність, нища швидкість і велика кількість девайсів, яка підтримує діапазон, тому і велика засміченість, а також нища ціна, а у сигналу 5Ghz менша дальність, вища швидкість і вища ціна обладнання.

2) NAT - це технологія, яка дозволяє дати кільком комп'ютерам одну ip адресу, і коректно налаштувати роботу в інтернеті.

3) Утиліта iptables - це утиліта для Linux. В цій утиліті можна працювати з трафіком, а також блокувати або дозволяти його.

4) TCP / IP - це два мережевих пpотокола Internet. Пpотокол TCP забезпечує встановлення надійного з'єднання між двома машинами і пеpедачі даних, а IP забезпечує маpшpутізацію мережевих пакетів. Перевага TCP / IP в тому, що можна об'єднати машини з різними ОС, і з NFS можна підключати диски з файлами іншої ОС.

5) Модель OSI - це протокол для розробки мережевих продуктів. Загалом виглядає як 7 рівнів. Зараз використовуюється стек протоколів TCP / IP.

6) Відмінності між TCP / IP та OSI це:
 1. TCP / IP має менше рівнів (в TCP / IP - 4, в OSI - 7)
 2. В TCP / IP можна пропускати рівні, що спрощує роботу.
 3. Деякі базові протоколи є частиною TCP / IP.

7) MAC-адреса - це унікальний ідентифікатор, який присвоюється кожному мережевому обладнанню, і дозволяє ідентифікувати кожну точку підключення.

8) Default gateway - це шлюз, який допомагає мережам з різними IP взаємодіяти між собою.
